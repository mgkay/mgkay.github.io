Matlog: Logistics Engineering Matlab Toolbox

The Matlab Toolbox Matlog has developed for use in the courses IE 453 and IE 754 at NCSU. The toolbox contains functions for solving facility location and logistics problems. 

Matlog Reference

Matlog Reference (HTML) provides a listing of all of the functions in the Matlog toolbox.

Installing Matlog

If connected to the NCSU network:

Before using the toolbox for the first time and before starting matlab, you should copy the file startup.m to your home directory. This file creates a path to the toolbox each time you start Matlab. Matlab and Matlog is accessible only from NCSU computers.

First time using toolbox, copy startup.m file to your home directory:

http://www.ise.ncsu.edu/kay/matlog/startup.m

Keep this file in you root K: directory; don't move it to a subdirectory because Matlog only looks into the root directory for the file when it starts.

Start Matlab: Matlab can be started from the icon in the Applications Window. A new window will appear for Matlab. The Matlab command prompt is ">>". Type helpwin to get a help window for all of the available Matlab toolboxes—click near the bottom to get matlog help.

If off-campus:

To use Matlab and Matlog on an off-campus computer, you will need a local version of Matlab (a Student Version of Matlab is available from the NCSU Bookstore) and you will need to copy a zipped file containing the Matlog directory to your local computer:

Matlog.zip (4.7 MB)

Updating Matlog: Individual files in Matlog will change on a regular basis (e.g., fixing errors). The file Matlog.zip is not updated as frequently. Instead, the Matlog function matlogupdate.m can be run to automatically download updates to Matlog files. You can add matlogupdate.m to your startup.m file to have it check for updates each time you start Matlab.
