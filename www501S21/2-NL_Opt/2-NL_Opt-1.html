1. Nonlinear Optimization¶

Nonlinear optimization refers to the problem of optimizing an objective function, where the elements of the objective function are points in a subset of Euclidean space. Unconstrained nonlinear optimization is when there are no constraints on the possible solutions, and when there are constraints, it is more commonly referred to as nonlinear programming, and linear programming is when the objective function and constraints are all linear. Mathematical programming refers to nonlinear and linear programming. Univariate optimization corresponds to the optimization of a function of a single variable on a bounded interval, and is the most basic type of nonlinear optimization, used as a building block in most nonlinear programming procedures.

Univariate Optimization¶

$ \begin{eqnarray} x^* &=& \mathrm{arg}\underset{x}{\operatorname{min}} \bigl\{ \, f(x) : L\!B \leq x \leq U\!B \bigr\} \\ T\!C^* &=& f(x^*) \end{eqnarray}$

2. User-Defined Functions¶

User-defined functions can be used to extend the capabilities of Julia beyond its basic functions. Although developing a set of functions to solve a particular problem is at the heart of using Julia to program, the easiest way to create a function is to do it in an incremental, scripting manner by writing each line, executing it, and, if it works, and only then copying it into the function.

Ex 1: Distance Function¶

Given a 1-D point x and m other 1-D points in p, create a function dist1 to determine the distance d from x to each of the m points in p:

$ \begin{array}{l} \quad\bf{d} = \left[ \begin{array}{c} \lvert x - p_{1} \rvert \\ \vdots \\ \lvert x - p_{m} \rvert \end{array} \right] \end{array} $

The best way to start is to create some example data for which you know the correct answer:

In [1]:
x = 3. 
p = [1, 2, 5, 9];

The first step is to subtract x from each row in p:

In [2]:
x .- p
Out[2]:
4-element Vector{Float64}:
  2.0
  1.0
 -2.0
 -6.0

Then take the absolute value of the result and assign the result to d:

In [3]:
d = abs.(x .- p)
Out[3]:
4-element Vector{Float64}:
 2.0
 1.0
 2.0
 6.0

A function can be created and then called to assign the result to d:

In [4]:
function dist1(x,p)
    return abs.(x .- p)
end
d = dist1(x,p)
Out[4]:
4-element Vector{Float64}:
 2.0
 1.0
 2.0
 6.0

The function dist1 can now be used inside of any expression; e.g., to calculate the sum of the distances from x to the points in p:

In [5]:
sum(dist1(x,p))
Out[5]:
11.0

One-Line Functions¶

One-line functions provide a convenient means to define simple functions. The function dist1 can be defined on a single line:

In [6]:
dist1(x,p) = abs.(x .- p)   # (One-line function)
sum(dist1(x,p))
Out[6]:
11.0

The sum of the distance from x to the points in p can be defined as the function sumdist1 with only x as an input argument and the current values of p used inside the function:

In [7]:
sumdist1(x) = sum(dist1(x,p))
sumdist1(3.)
Out[7]:
11.0
In [8]:
sumdist1(8.)
Out[8]:
17.0

An array comprehension can be used to create an array of outputs from sumdist1:

In [9]:
[sumdist1(x) for x in [3., 8.]]
Out[9]:
2-element Vector{Float64}:
 11.0
 17.0
In [10]:
[sumdist1(x) for x in p]
Out[10]:
4-element Vector{Int64}:
 13
 11
 11
 19

Although the sum of the distances sumdist1 could be defined directly without using dist1; but, by first defining dist1, it allows other distance-related functions to be defined in addition to summation:

In [11]:
maxdist1(x) = maximum(dist1(x,p))    # Max distance from x to any point in p
maxdist1(3.)
Out[11]:
6.0
In [12]:
maxdist1(8.)
Out[12]:
7.0
In [13]:
sum2dist1(x) = sum(dist1(x,p).^2)    # Sum of squared distances
sum2dist1(3.)
Out[13]:
45.0

3. Minimization¶

The function optimize performs general-purpose nonlinear minimization. Note: maximization corresponds to the minimization of the negative of the function to be maximized.

Ex 2: Minimize the Sum of Squared Distances¶

Given a bounded search interval [LB, UB], determine a value for x that minimizes the sum of squared distances to each point in p. In this case, the centroid/center-of-gravity/mean of p corresponds to the point that minimizes the sum of squared distances:

In [15]:
using Statistics
mean(p)
Out[15]:
4.25

This can also be determined using optimize, with the advantage that is can also minimize other functions where there is not a known analytical solution:

In [16]:
LB = 0; UB = 10
using Optim
optimize(sum2dist1, LB, UB)
Out[16]:
Results of Optimization Algorithm
 * Algorithm: Brent's Method
 * Search Interval: [0.000000, 10.000000]
 * Minimizer: 4.250000e+00
 * Minimum: 3.875000e+01
 * Iterations: 5
 * Convergence: max(|x - x_upper|, |x - x_lower|) <= 2*(1.5e-08*|x|+2.2e-16): true
 * Objective Function Calls: 6

To assign the optimal location and minimum total distance values to variables:

In [17]:
res = optimize(sum2dist1, LB, UB)
x = res.minimizer
TD = res.minimum
@show x TD;

x = 4.25
TD = 38.75

To return only the optimal location :

In [18]:
x = optimize(sum2dist1, LB, UB).minimizer
Out[18]:
4.25

Can then get the minimimum total distance if needed:

In [19]:
TD = sum2dist1(x)
Out[19]:
38.75
In [20]:
plot(sum2dist1,LB,UB, legend=false, xticks=p)
scatter!([x],[sum2dist1(x)])
Out[20]:

Anonymous Functions¶

In order not to have to first define a function get the sum, an anonymous function can be used instead of a named function when the result is only going to be used once as an input argument inside of another function:

In [22]:
x = optimize(x -> maximum(dist1(x,p)), LB, UB).minimizer
Out[22]:
5.000000000000001

Ex 4: Maximize the Minimum Distance¶

Trying to determine a location x between between 0 and 10 that is as far away as possible from any of the points 1, 2, 5, and 9 in p. This corresponds to trying to maximize the minimum distance of x from any point in p.

In [23]:
mindist1(x) = minimum(dist1(x,p))
x = optimize(x -> -mindist1(x), LB, UB).minimizer
plot(mindist1,LB,UB, legend=false, xticks=p)
scatter!([x],[mindist1(x)])
Out[23]:

Beware!: Can see that the global maximum of 2 at 7 was not found. Can cut off local maximum of 1.5 at 3.5 by increasing LB to 4:

In [24]:
x = optimize(x -> -mindist1(x), 4, UB).minimizer
plot(mindist1,LB,UB, legend=false, xticks=p)
scatter!([x],[mindist1(x)])
Out[24]:
